# Happy Voyager

Happy Voyager is a modern travel and lifestyle website designed to inspire, inform, and empower travelers. The platform features engaging blogs, destination guides, travel tips, and resources to help users plan memorable journeys. Built with a focus on performance, accessibility, and beautiful UI, Happy Voyager leverages the latest web technologies for a seamless user experience.

## Features

- Travel blogs and stories
- Destination highlights and guides
- Contact and lead forms
- FAQs and testimonials
- Responsive, mobile-friendly design

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Language:** TypeScript
- **UI Library:** [Tailwind CSS](https://tailwindcss.com/)
- **Styling:** CSS Modules, Tailwind
- **Other:** React, modern best practices

## Getting Started

### 1. Clone the repository

```bash
git clone <repo-url>
cd happy-voyager
```

### 2. Install dependencies

Using npm:

```bash
npm install
```

Or with yarn:

```bash
yarn install
```

Or with pnpm:

```bash
pnpm install
```

### 3. Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.

## Project Structure

- `app/` — Main application pages and routes
- `components/` — Reusable UI components
- `lib/` — Utilities and mock data
- `public/` — Static assets
- `utils/` — API utilities

## Customization

You can start editing the homepage by modifying `app/page.tsx`. The site supports hot reloading for rapid development.
